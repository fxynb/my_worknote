# 字符

## 普通字符
普通字符包括没有显示指定为元字符的所有可打印和不可打印字符。这包括所有的大写和小写字母，所有数字，所有标点符号以及其他符号

## 非打印字符
\cx 匹配由x指明的控制字符，例如，\cM 匹配一个control-M或回车符

## 元字符
.  除换行符以外的任何字符

\w 字符串数字下划线中文

\d 匹配数字

\s 空白字符

\b 单词的开始或者结束

^ 字符串的开始

$ 字符串的结尾

## 重复限定符

* 重复零次或更多次

+ 重复一次或更多次

？重复零次或一次

{n} 重复n次

{n,} 重复n次或多次

{n,m} 重复n到m次


## 修饰符

/g 全局匹配 查找到所有的配项                                                                                                                                                                                                                                                                                                    /i 忽略大小写

/m 多行匹配

/s 匹配包含换行符
